"Drawing Drawing Drawing": Experiment 6 - Text Art

Imitate

For this project I was inspired by this example: http://www.generative-gestaltung.de/2/sketches/?01_P/P_2_3_3_01. I liked the idea of using text as a brush to paint with, and it gave me the idea to try to make the text more connected to the image being drawn.

Integrate

To start, I wanted to add the ability to select colors. I added 8 color options other than black. Red, orange, yellow, green, blue, purple, pink, and light gray.

After experimenting with the colors, I decided to add light blue as well.

Then, I set it so the text varied based on the color, so that the text would describe the colors being used.

Overall, I think that turned out pretty well!

Innovate

At this point, I wanted to make my project more unique and distinct. Since it is fairly simple, I began with the font. I set it such that the font changes at the end of each word.

I like how that turned out! I decided to add a way for the user to change the minimum font size.

I felt that being able to change the font size would allow for the user to create more varied drawings. Overall, I’m happy with how this project turned out!
